Convert PFX to PEM

Convert a PFX (PKCS12) Certificate to the PEM format

18 Nov 2016, 20:46
x509 / openssl

Windows uses the PFX file format for certificates, and UNIX uses the PEM format. If you have to use both platforms you frequently have to convert between the two worlds. PEM is a good base format choice. Store in PEM, and then convert to PFX as necessary.

openssl pkcs12 -in cert.pfx -out cert.pem -nodes

The -nodes does not encrypt the private key.
